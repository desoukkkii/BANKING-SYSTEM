Banking System – Goldtrust Bank

Description:

    A modern, fully functional digital banking interface that allows users to deposit, withdraw, and transfer funds. Features real-time balance updates, transaction history with timestamps, and a clean, responsive UI. Built with HTML, CSS, and vanilla JavaScript. Data persists during the session, and users can reset to default state.

Tech Stack: HTML, CSS, JavaScript, Font Awesome

Key Features:

    Deposit, withdraw, and transfer money

    Real-time balance updates

    Transaction history with timestamps

    Reset functionality

    Responsive design
